\doxysubsection{Detailed Description}
\begin{DoxyNote}{Note}
x\mbox{[}i,j,t,k\mbox{]} is a 4D binary variable to represent the route decisions. It is equal to 1 if it is true that we are going from node i to node j at time t with vehicle k. 
\end{DoxyNote}
